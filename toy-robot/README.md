# Toy Robot

Program accepts input from `stdin` as a list of commands separated by new lines  
Output is redirected to `stdout`

## Run examples:

Example A:
```
sbt run < examples/example_a.txt
```
Output:
```
0,1,NORTH
```

Example B:
```
sbt run < examples/example_b.txt
```
Output:
```
0,0,WEST
```

Example C:
```
sbt run < examples/example_c.txt
```
Output:
```
3,3,NORTH
```
